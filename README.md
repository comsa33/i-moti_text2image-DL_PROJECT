## text-to-image-DL_PROJECT
-	텍스트를 입력하면 해당 텍스트 의미에 해당하는 적절한 이모티콘을 생성해내는 모델 개발 프로젝트
-	DALL-e 와 StackGAN 등의 text-to-image 모델들의 구조보다 좀 더 경량화된 커스텀 신경망 구축 프로젝트
